# Что такое DNS и как его настроить?

## Простыми словами

**DNS** — это как телефонная книга для интернета. Когда вы вводите в браузере `mysite.com`, DNS говорит браузеру: "Этот сайт находится по адресу 123.45.67.89".

## Зачем это нужно?

Если у вас есть домен (например, `mysite.com`), вы хотите, чтобы люди могли открыть ваш сайт, введя это имя, а не длинный IP-адрес типа `123.45.67.89`.

## Как настроить DNS (пошагово)

### Шаг 1: Узнайте IP-адрес вашего сервера

На вашем сервере выполните команду:
```bash
curl ifconfig.me
```

Или:
```bash
hostname -I
```

Скопируйте полученный IP-адрес (например: `123.45.67.89`)

### Шаг 2: Зайдите в панель управления доменом

Где вы покупали домен, там и настраивается DNS. Обычно это:
- **reg.ru** → Личный кабинет → DNS-серверы / Управление DNS
- **timeweb.com** → Панель управления → Домены → DNS-зона
- **namecheap.com** → Domain List → Manage → Advanced DNS
- **godaddy.com** → Мои продукты → DNS

### Шаг 3: Добавьте A-запись

Найдите кнопку "Добавить запись" или "Add Record" и заполните:

| Поле | Значение | Пример |
|------|----------|--------|
| **Тип** | `A` | A |
| **Имя/Хост** | `@` (или оставьте пустым) | @ |
| **Значение/IP** | IP вашего сервера | 123.45.67.89 |
| **TTL** | Оставьте по умолчанию | 3600 |

**Что означает `@`?**  
`@` — это символ для основного домена. Если ваш домен `mysite.com`, то `@` означает именно `mysite.com`.

### Шаг 4: Опционально — добавьте www

Если хотите, чтобы `www.mysite.com` тоже работал:

| Поле | Значение | Пример |
|------|----------|--------|
| **Тип** | `A` | A |
| **Имя/Хост** | `www` | www |
| **Значение/IP** | Тот же IP сервера | 123.45.67.89 |
| **TTL** | Оставьте по умолчанию | 3600 |

### Шаг 5: Сохраните и подождите

- Нажмите "Сохранить" или "Add Record"
- Подождите **5-30 минут** (иногда до часа) — DNS изменения распространяются не мгновенно

### Шаг 6: Проверьте

На вашем компьютере выполните:
```bash
ping mysite.com
```

Должен показать IP вашего сервера. Если показывает — всё готово!

## Если у вас нет домена

Можно использовать IP-адрес напрямую:
- `http://123.45.67.89/` — просто вставьте IP вашего сервера

## Частые вопросы

**Q: Сколько времени нужно ждать?**  
A: Обычно 5-30 минут, иногда до часа.

**Q: Что делать, если не работает?**  
A: 
1. Проверьте, что IP-адрес правильный
2. Убедитесь, что сохранили изменения
3. Подождите еще немного
4. Проверьте через `ping yourdomain.com`

**Q: Нужно ли что-то настраивать в Docker?**  
A: Нет! Nginx уже настроен для работы с любым доменом. Просто настройте DNS и всё заработает.
